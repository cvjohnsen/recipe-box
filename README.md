# My Recipe Box

My Recipe Box is a React application that allows users to save, organize, edit, delete, favorite, and filter recipes.

## Features
- Add new recipes
- Edit existing recipes
- Delete recipes
- Mark recipes as favorite
- Filter by category
- Show favorite recipes only
- Save data with Local Storage

## Technologies Used
- React
- Vite
- React Router DOM
- CSS

## How to Run
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`

## Notes
Recipe data is stored in the browser using Local Storage.
